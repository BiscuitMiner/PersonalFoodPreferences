# Personal Food Preferences 文檔任務清單

> PackageId: `BiscuitMiner.PersonalFoodPreferences`
> 目標：建立面向未來更新、維護、相容補丁與其他 MOD 聯動的快速查閱文檔。
> 必須遵守：`AGENTS.md` | 每輪對話僅集中輸出一份說明文檔

---

## 文檔工作規則

- 每輪只新增或重寫一份 `.md` 文檔。
- 文檔優先服務「維護查閱」與「聯動開發」，不是玩家宣傳文。
- 每份文檔必須能獨立回答：入口在哪、資料在哪、狀態存在哪、如何驗證、擴展時不能碰什麼。
- 機制文檔不添加 `## 實作 + 驗證` 段落；驗證內容集中放在文檔內的驗證 / 排查段落。
- 完成任何文檔後，必須同步更新本 `TASKS.md` 對應任務勾選狀態。
- 不在文檔任務中修改 C#、XML、DLL，除非使用者另開實作任務。
- 若文檔發現代碼或資料缺陷，只記錄為後續修正任務，不在同輪混入修正。
- 涉及原版入口時，需列出原版 class / method / Def 關鍵字。
- 涉及 SoC 時，需明確分開 Patch、CoreLogic、Utility、ThingComp / HediffComp、XML / Settings 的職責。
